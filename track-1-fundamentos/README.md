# Track 1 — Fundamentos

Material do track de fundamentos do curso DeerFlow 2.0.

**Público**: devs curiosos, PMs técnicos, pesquisadores. Não exige Python avançado.

**Entregável**: ao final do track, o aluno consegue rodar, configurar e usar o DeerFlow para tarefas reais sem precisar tocar código.
