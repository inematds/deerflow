# Curso: DeerFlow 2.0 — do básico ao avançado

Repositório de conteúdo do curso sobre **DeerFlow 2.0**, o super agent harness open-source da ByteDance.

O curso cobre três frentes:

1. **Track 1 — Fundamentos**: o que é, como instalar, como usar no dia a dia (público: devs curiosos, PMs técnicos, pesquisadores).
2. **Track 2 — Avançado**: arquitetura interna, criação de skills, sub-agents, MCP servers, embedding, contribuição upstream (público: engenheiros de IA, platform engineers).
3. **Comparativo**: DeerFlow vs. Claude Code vs. Antigravity — quando usar o quê.

## Como usar este repo

- O arquivo `docs/plano-do-curso.md` é o ponto de partida. Ele contém o índice completo, objetivos, labs e entregáveis de cada módulo.
- Cada track tem seu próprio diretório com o conteúdo das aulas (roteiros, slides-chave, código dos labs).
- O diretório `comparativo/` hospeda o material transversal de comparação entre plataformas.
