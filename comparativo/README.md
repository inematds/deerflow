# Módulo transversal — Comparativo

DeerFlow vs. Claude Code vs. Antigravity — **quando usar o quê**.

Pode rodar depois do Módulo 3 do Track 1 ou como aula standalone. A ideia é dar clareza de posicionamento sem cair em "X é melhor que Y".

## Conteúdo previsto

- **Framework de comparação** — matriz fixa de 10 eixos (categoria, execução, extensibilidade, memória, etc.)
- **Posicionamento de cada plataforma** — esqueleto honesto a validar com os alunos
- **Labs comparativos** — 4 tarefas rodando lado a lado nas 3 plataformas:
  1. Tarefa de código (refatorar módulo Python + abrir PR)
  2. Tarefa de research (relatório com fontes citadas)
  3. Tarefa híbrida (análise de CSV → PPT → post no Slack)
  4. Tarefa de extensibilidade (criar skill/tool nova em cada plataforma)

**Entregável**: o aluno sabe escolher a ferramenta certa por tipo de tarefa, não por moda.

Ver o plano completo em [`../docs/plano-do-curso.md`](../docs/plano-do-curso.md).
