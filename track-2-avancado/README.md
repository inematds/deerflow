# Track 2 — Avançado

Material do track avançado do curso DeerFlow 2.0.

**Público**: engenheiros de IA, platform engineers, contribuidores upstream.

**Pré-requisito**: Track 1 concluído + Python 3.12 confortável + noção de LangGraph/LangChain.

**Entregável**: ao final do track, o aluno consegue estender o harness (skill nova, custom agent, MCP server, provider), embutir o DeerFlow em outro sistema e contribuir upstream.
